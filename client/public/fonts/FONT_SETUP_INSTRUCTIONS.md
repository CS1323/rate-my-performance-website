# SF Collegiate Font Setup

## Manual Download Instructions

1. Go to: https://www.fontsquirrel.com/fonts/SF-Collegiate
2. Download the "Solid Regular" font file
3. Extract the TTF file to this directory (fonts/)
4. The TTF file should be named: `sf-collegiate.ttf`

Once the font file is in place, the @font-face rule in Header.css will automatically load it.
